# Team Sports Course

## 📖 Course Overview

This course develops essential teamwork, leadership, and athletic skills through structured sports activities and theoretical learning.

### Learning Objectives
- Develop effective teamwork and communication skills
- Learn and apply leadership principles
- Understand fitness, training, and athletic performance
- Build character through sports
